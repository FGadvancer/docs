### SearchGroupsParam

| 字段Parameter Name                | 字段Type                                                  | Description                                      |
| ---------------------- | --------------------------------------------------------- | ----------------------------------------- |

|IsSearchGroupID | bool  |      	是否以关键词搜索Group ID(注：两个不可以同时为 false)，默认 false                 |
