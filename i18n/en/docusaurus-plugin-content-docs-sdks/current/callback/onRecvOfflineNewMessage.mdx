# onRecvOfflineNewMessage

## Feature Introduction

:::info

When the application is running in the background and a new message is received, this callback is triggered. The callback will carry only one message.

:::

### Example Call

```js showLineNumbers
import { getSDK, CbEvents } from 'open-im-sdk-wasm';
// or
// import { OpenIMSDK, CbEvents } from 'open-im-sdk';
// const IMSDK = new OpenIMSDK();
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnRecvOfflineNewMessage, ({ data }) => {
  // data is the new offline message
});
```
